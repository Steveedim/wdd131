Site Plan

Site name

Calabar Tourist Site

This site represents a website dedicated to showcasing the culture of Cross River State and their tourist sites

Optional domain available: calabartouristsite.com

Site Purpose

The purpose of this website is Serve as educational resource for users who want to learn more about Cross River divers cultural heritage and explore tourist attraction across the state and the world at large.

Scenarios

  • Scenario 1: User wants to learn about Calabar Carnival
  • Scenario 2: User is looking for tourist attractions in Cross River
  • Scenario 3: User wants to explore cultural heritage sites

Color Schema

  • #375E97 - Primary color for headers and button
  • #FB6542 - Secondary color for accents and highlights
  • #FFBB00 - Tertiary color for backgrounds and borders

Typography

  • Headings: 'Roboto', sans-serif
  • Body Text: 'Tagesschrift', serif
